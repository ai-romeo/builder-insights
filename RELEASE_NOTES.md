## 🚀 What's New in v2.0

### Core Features

1. **Value Scoring (0-10)**
   - Information density (40%)
   - Originality (35%)
   - Engagement quality (25%)

2. **Auto-Categorization**
   - 🏗️ Product Launch
   - 🔬 Technical Breakthrough
   - 📊 Industry Insight
   - 💰 Funding & M&A

3. **33 Builders (8 Chinese)**
   - Global: Karpathy, Sam Altman, Replit CEO, etc.
   - Chinese: 李开复，吴恩达，王小川，杨植麟 (Kimi), etc.

4. **Bilingual Support**
   - English + Chinese
   - Configurable language preference

5. **Improved Content Fetch**
   - GitHub Feed (stable)
   - Rettiwt API (no auth)
   - Supadata for YouTube
